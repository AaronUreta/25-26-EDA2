# 🚀 Reto de Clase: Trazabilidad en Algoritmos de Ordenación

Este proyecto implementa **trazas de ejecución detalladas** sobre algoritmos clásicos de ordenación para visualizar la toma de decisiones y la evolución del estado de los datos en tiempo real.

### 🔍 Objetivos del Reto
* **Visibilidad Total:** Reconstruir la secuencia de transformaciones del array exclusivamente a partir de la salida por consola.
* **Visualización Recursiva:** Uso de indentación proporcional para mapear el árbol de llamadas en algoritmos como *Quick Sort* y *Merge Sort*.
* **Análisis de Decisiones:** Registro exhaustivo de cada comparación, intercambio y caso base alcanzado.

### 🛠️ Restricción de Inalterabilidad
Un pilar fundamental de este desarrollo es el respeto a la **lógica original**. Se ha mantenido la estructura de control intacta, enriqueciendo el código únicamente con instrucciones de salida y métodos auxiliares de formato, sin alterar el comportamiento algorítmico.

### 📊 Escenario de Prueba
Todas las implementaciones han sido validadas utilizando el conjunto de datos obligatorio:
`int[] array = {5, 2, 8, 1, 9, 3};`
